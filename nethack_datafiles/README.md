# NetHack Runtime Data Files (RV32 Generated)

This directory contains runtime data files generated by running NetHack utilities 
(makedefs, lev_comp, dgn_comp) in the RV32 simulator.

These files are architecture-specific and generated once, then reused to avoid 
expensive regeneration in the simulator (~50M+ instructions per tool run).

## Structure
```
nethack_datafiles/
└── usr/games/lib/nethackdir/  # Standard NetHack runtime directory
    ├── data                    # Main game data (205KB) - makedefs -d
    ├── rumors                  # Rumor file - makedefs -r
    ├── oracles                 # Oracle sayings - makedefs -h
    ├── quest.dat               # Quest text - makedefs -q
    ├── options                 # Options file - makedefs -v
    ├── dungeon                 # Dungeon structure - dgn_comp
    └── *.lev                   # 113 level files - lev_comp
```

## Generation Info
- Generated by: RV32IM simulator (pyrv32)
- Utilities built with: picolibc, riscv64-unknown-elf-gcc
- See: NETHACK_UTIL_FILES.md for complete generation commands and metrics
